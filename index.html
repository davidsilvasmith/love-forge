If one partner is skeptical

Use this 30-second opener to lower resistance: "I don't want to do a huge relationship overhaul. Can we test one 10-minute tool for 7 days, then keep it only if it helps both of us?"

  • Start with low-friction tools: Pause-and-Return Tool or Daily 15-Minute Connection Block.
  • Keep it practical: one tool, one week, one score (1-10 connection).
  • If scores do not improve, swap tools instead of arguing about the process.

  • Our arguments escalate fast → Fight Fair Protocol, Pause-and-Return Tool

Choose by the time you have tonight

  • 5 minutes → Pause-and-Return Tool (de-escalate and set a return time)
  • 10 minutes → Distance Connection Protocol (quick emotional reconnect script)
  • 15 minutes → Daily 15-Minute Connection Block (structured check-in)
  • 25 minutes → Fight Fair Protocol (repair after conflict)

Track one number: After each tool, both partners rate connection from 1-10. If the average improves by 2+ points in 7 days, keep the tool.

7-day quick plan (copy this)

  1. Day 1: Run the Disconnection Detector together and pick one trigger to focus on.
  2. Days 2-6: Do the Daily 15-Minute Connection Block at the same time each day.
  3. Any conflict spike: Pause with the Pause-and-Return Tool, then resume when calm.
  4. Day 7: Use the Monthly Marriage Retro and decide: keep, tweak, or replace the routine.

Success check: If your average connection score moves up by 2+ points, keep this plan for another week.

Before you pick a tool

  • If there is physical intimidation, threats, or ongoing emotional abuse, skip DIY tools and involve a licensed professional now.
  • If trust was broken recently (affair, hidden debt, repeated lying), start with the Third-Party Escalation Tool before trying communication scripts.
  • If talks escalate repeatedly, use the Pause-and-Return Tool first, then come back to deeper tools.

Goal: pick the safest next step, not the hardest conversation.
