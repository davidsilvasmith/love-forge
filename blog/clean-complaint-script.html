Clean Complaint Script

For busy parents • practical in 10-15 minutes

🧼 Cleaner asks
🧯 Fewer blame spirals
🛠️ Faster repair
📈 Better connection score
Problem

Needs come out as blame

Under kid + work pressure, complaints sound like attacks and trigger defensiveness.

Promise

One script that lowers heat fast

Use a simple sentence frame so you can ask clearly without starting a fight.

Quick start block (tonight)

  1. Set a 10-minute timer.
  2. Use this frame exactly: “When X happened, I felt Y, I need Z.”
  3. One complaint at a time (no history dump).
  4. End with one concrete step for tomorrow.

Copy/paste script: “I don’t want a giant relationship talk. I want one small win tonight. Can we run the Clean Complaint Script for 10 minutes and see if it helps?”

Quick action checklist (easiest version)

  1. Pick one tiny step from this tool for tonight.
  2. Do it in under 10 minutes.
  3. Text or tell your partner one clear next step for tomorrow.
  4. Rate helpfulness 1-10 before bed and keep only what works.
