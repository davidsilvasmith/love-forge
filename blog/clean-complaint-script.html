Clean Complaint Script

For busy parents • practical in 10-15 minutes

🧼 Cleaner asks
🧯 Fewer blame spirals
🛠️ Faster repair
📈 Better connection score
Problem

Needs come out as blame

Under kid + work pressure, complaints sound like attacks and trigger defensiveness.

Promise

One script that lowers heat fast

Use a simple sentence frame so you can ask clearly without starting a fight.

Do this tonight (one script, 10 minutes)

First step (do this now): Text this exactly: “Can we do a 10-minute clean complaint at 8:30? I’ll keep it short and specific.”

Copy/paste script to use in the talk:
“When [specific moment] happened, I felt [one feeling]. I need [one specific change] tomorrow. Are you open to that?”

  1. Minute 0-1: Set a 10-minute timer and say: “One issue only. No old stuff.”
  2. Minute 1-3: Partner A reads the script once. No extra context.
  3. Minute 3-5: Partner B mirrors: “I heard you say ___, and you need ___ tomorrow. Did I get it?”
  4. Minute 5-7: Partner B answers yes/no on the request. If no, offer one doable alternative.
  5. Minute 7-9: Switch roles and repeat with one issue.
  6. Minute 9-10: Lock one concrete action for tomorrow (who, what, when).

Low-energy version (3 minutes): Run one round only (A speaks, B mirrors, one tomorrow action).

If emotions spike: Say “Pause. I want to come back at 9:15.” Then use Pause-and-Return Tool.

How to use this tool effectively

  • When to use it: Use Clean Complaint Script when this exact problem is happening now, not as a general reading exercise.
  • How often: Run it once today, then repeat 3-7 times before judging results.
  • Common mistake: Changing multiple things at once. Use one tool at a time so you can see what works.
