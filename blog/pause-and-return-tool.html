14. Pause-and-Return Tool

Problem-first framework

Problem

Conversations escalate past productive and end in shutdown, stonewalling, or regret.

When to call pause

  • Raised voices, contempt, repeated interruptions
  • Heart racing / overwhelm / urge to "win"
  • One person is emotionally flooded

How to run it

  1. Say: "Pause. I care about this. I’ll return at [specific time]."
  2. Take 20-60 minutes (walk, breathe, hydrate, no doom-scroll).
  3. Return on time and restart with one-sentence goal.
  4. Use clean complaint format for the first 2 minutes.

What not to do on pause

  • Don’t text-argue during break
  • Don’t recruit allies/friends mid-fight
  • Don’t disappear with no return time

Copy/paste script

"I’m getting flooded and I don’t want to hurt us. Pause now, return at 8:30, and we’ll solve one issue."

Success metric

Track 2 weeks: fewer blowups, shorter fights, faster repair. If yes, make this a permanent rule.
