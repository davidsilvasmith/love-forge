Weekly State-of-Us Scorecard

For busy parents • practical in 10-15 minutes

Quick start (tonight)

⏱️ Set a 10-minute timer 📊 Score 4 areas 🗣️ Say one short opener ✅ Choose one tiny next step

Goal: leave feeling more connected, not drained.

Problem

You know something feels off

When life is a blur, this helps you name the one thing to fix first.

Best use case

When you keep having the same stress loop

This gives you one clear fix for tomorrow.

Time cost

10–15 minutes total

Do it after bedtime, in the car line, or on a short walk.

Score these 4 areas (1-10)

Category 1

Trust

Did I feel emotionally safe with you this week?

Category 2

Closeness

Did we feel close, or just run logistics all week?

Category 3

Teamwork

Did house and kid stress feel shared and fair?

Category 4

Fun

Did we laugh or have one good moment together?

Copy/paste script

“I love us, and I don’t want a big heavy talk tonight. Can we do this 10-minute scorecard and pick one small win?”

Kid-chaos fallback

  • Do it in two 5-minute rounds if kids interrupt.
  • If either person is too upset, use Pause-and-Return first.
  • No time? Score just two categories tonight.

What to do with your score

8-10

Keep doing what works

Protect your routines. Repeat next week.

5-7

Pick one weak spot

Choose one issue and use one tool this week.

1-4

Go into repair mode

Pause blame. Focus on safety and one small agreement.

Simple rule: Keep tools that raise your score by 2 or more points. Drop the rest.

Starter kit (no guesswork)

Pick one of these and run it exactly tonight:

  1. 5-minute version: Read the copy/paste script out loud, then do one tiny action from this tool immediately.
  2. 10-minute version: Set a timer, follow the “Do this tonight” steps top to bottom, and stop when timer ends.
  3. Couple-text version: Send this now: “Can we do Weekly State-of-Us Scorecard tonight for 10 minutes? I already picked the first step so this is easy.”

If you feel stuck, use the 5-minute version first. Progress beats perfection.

How to use this tool effectively

  • When to use it: Use Weekly State-of-Us Scorecard when this exact problem is happening now, not as a general reading exercise.
  • How often: Run it once today, then repeat 3-7 times before judging results.
  • Common mistake: Changing multiple things at once. Use one tool at a time so you can see what works.
  • If it feels awkward: Use the script exactly once, keep the first run under 10 minutes, and rate helpfulness 1-10.
