6. Daily 15-Minute Connection Block

Problem-first framework

Problem

You talk all day, but it’s mostly logistics. The relationship starts feeling efficient, not close.

Why this happens

  • Stress pushes every conversation into task mode.
  • Both partners wait to feel close before they act close.
  • No recurring ritual means connection only happens “if there’s time.”

10-minute setup (do this once)

  1. Pick one fixed time daily (after dinner, before bed, or after kids down).
  2. Phones face down and out of reach.
  3. Use timer for 15 minutes so nobody feels trapped.

How to run it (15 minutes)

  1. Minute 1-2: One sentence check-in each: “Today I’m feeling ___.”
  2. Minute 3-10: One person shares; the other only reflects back.
  3. Minute 11-13: Switch roles.
  4. Minute 14-15: One appreciation + one small ask for tomorrow.

Copy/paste script

"I miss feeling close to you. Can we try 15 minutes a day this week—no phones, no logistics, just us?"

"What I hear you saying is ____. Did I get that right?"

Rules (non-negotiable)

  • No fixing unless asked: “Do you want comfort or solutions?”
  • No scorekeeping (“I did this yesterday”).
  • No conflict topics in this block unless both agree.

Success metric

Each day rate closeness 1-10. After 7 days, keep this tool if average score improves by 2+ points.

If it fails

Shorten to 8 minutes for 3 days, then ramp back up. Consistency beats intensity.
