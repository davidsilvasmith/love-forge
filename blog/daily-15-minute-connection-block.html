Daily 15-Minute Connection Block

For busy parents • practical in 10-15 minutes

💬 15-minute script 📵 No phones ⏱️ Timer keeps it light 📈 Score nightly
Problem

Logistics replaced connection

Most nights become school, meals, and schedules. Emotional connection gets postponed.

Goal

Create one reliable daily rep

Protect a short check-in that keeps stress from turning into distance.

Constraint

Keep it simple enough to repeat

No giant processing session. One timer, one script, one next step.

Quick start (tonight)

1️⃣

Pick a time

Choose a protected 15-minute block after dinner or bedtime.

2️⃣

Run the script

Start a timer and read the opener exactly. No improv under stress.

3️⃣

Close with one next step

End with one tiny action for tomorrow and score connection 1-10.

Use this exact plan (copy this)

  1. Minute 0-2: Name today’s focus in one sentence.
  2. Minute 2-6: Each person gets 2 minutes uninterrupted.
  3. Minute 6-9: Choose one action for next 24 hours.
  4. Minute 9-10: Confirm when you’ll check in again.

Copy/paste script

“I don’t want a giant relationship talk. I want one small win tonight. Can we try the Daily 15-Minute Connection Block for 10 minutes and see if it helps?”

How to use this tool effectively

  • When to use it: Use Daily 15-Minute Connection Block when this exact problem is happening now, not as a general reading exercise.
  • How often: Run it once today, then repeat 3-7 times before judging results.
  • Common mistake: Changing multiple things at once. Use one tool at a time so you can see what works.
  • If it feels awkward: Use the script exactly once, keep the first run under 10 minutes, and rate helpfulness 1-10.
