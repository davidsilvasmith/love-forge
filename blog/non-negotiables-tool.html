Non-Negotiables Tool

For busy parents • practical in 10-15 minutes

🧭 Value alignment ⏱️ 10-15 minutes 🧱 Clear boundaries ❤️ Less repeat conflict
Problem

Hidden dealbreakers keep colliding

Unspoken priorities create repeat fights that feel bigger every week.

Goal

Name your top 3 each

Get concrete fast so you can stop guessing and start coordinating.

Constraint

Keep it short and safe

Timer on, one script, one next step. No courtroom debate tonight.

Quick start (tonight)

📝

Write top 3 privately

Each person lists their 3 non-negotiables in 2 minutes. No debate yet.

👂

Read, don’t react

Share lists straight through. No interrupting, defending, or correcting.

🔍

Mark overlap + friction

Circle one overlap and one tension point so the next move is concrete.

📌

Lock tomorrow’s action

Pick one small action for tomorrow that protects both people’s must-haves.

Copy/paste script

“I don’t want a giant relationship talk. I want one small win tonight. Can we try the Non-Negotiables Tool for 10 minutes and see if it helps?”

When exhausted

Lite mode

Share only your top 1 non-negotiable each and set one boundary for tomorrow.

When interrupted

Split mode

Do 5 minutes now and 5 minutes after bedtime. Don’t scrap the session.

When flooded

Reset mode

Run pause-and-return first, then come back when both nervous systems are calm.

Sample agenda (copy this)

  1. Minute 0-2: Name today’s focus in one sentence.
  2. Minute 2-6: Each person gets 2 minutes uninterrupted.
  3. Minute 6-9: Choose one action for next 24 hours.
  4. Minute 9-10: Confirm when you’ll check in again.
