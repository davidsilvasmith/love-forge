Starter kit (no guesswork)

Pick one of these and run it exactly tonight:

  1. 5-minute version: Read the copy/paste script out loud, then do one tiny action from this tool immediately.
  2. 10-minute version: Set a timer, follow the “Do this tonight” steps top to bottom, and stop when timer ends.
  3. Couple-text version: Send this now: “Can we do Non-Negotiables Tool tonight for 10 minutes? I already picked the first step so this is easy.”

If you feel stuck, use the 5-minute version first. Progress beats perfection.

How to use this tool effectively

  • When to use it: Use Non-Negotiables Tool when this exact problem is happening now, not as a general reading exercise.
  • How often: Run it once today, then repeat 3-7 times before judging results.
  • Common mistake: Changing multiple things at once. Use one tool at a time so you can see what works.
  • If it feels awkward: Use the script exactly once, keep the first run under 10 minutes, and rate helpfulness 1-10.

Tomorrow rep + fallback + escalation
