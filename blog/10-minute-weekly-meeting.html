5. 10-Minute Weekly Meeting

Problem-first framework

Problem

Weeks pass without intentional relationship maintenance, so tiny issues pile up.

How to run (10 minutes, same day every week)

  1. 2 min: Win of the week (“One thing you did that helped us”).
  2. 3 min: Friction point (“One thing that felt off”).
  3. 3 min: One request each for next week.
  4. 2 min: Calendar one connection moment (date walk, coffee, call).

Rules

  • Short and structured (set a timer).
  • No character attacks.
  • End with one appreciation every time.

Copy/paste opener

"Can we do our 10-minute weekly check-in now? One win, one friction, one request each."

Success metric

Track weekly: unresolved tensions count. If it trends down over 3 weeks, keep this tool permanently.
