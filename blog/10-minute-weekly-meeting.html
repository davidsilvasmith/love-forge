10-Minute Weekly Meeting

For busy parents • highest-leverage weekly habit

Why this works

Quick relationship check-ins work best when they are short, structured, and repeated. Research-backed check-in formats consistently emphasize three things: appreciation, one manageable issue, and one concrete next step.

💛 Start positive 🎯 One issue only ✅ One action by tomorrow

Use this exact agenda (copy this)

  1. Minute 0–2: Appreciation. Each person shares one specific thing they appreciated this week.
  2. Minute 2–5: Temperature check. Rate connection 1–10 and name one tension point (no blame, just facts).
  3. Minute 5–8: Solve one thing. Pick one issue only. Use: “When X happened, I felt Y, and I need Z.”
  4. Minute 8–10: Lock the plan. Agree one action each for the next 7 days and put it on calendar now.

If it starts going sideways

  • Use one-issue rule (don’t kitchen-sink).
  • If emotions spike, switch to Pause-and-Return.
  • Return the same day and finish the final 2 minutes.

Success metric (simple)

Track one number weekly: Connection Score (1–10). If it rises by 2+ after 3 weeks, keep this meeting permanently.

How to use this tool effectively

  • When to use it: Use 10-Minute Weekly Meeting when this exact problem is happening now, not as a general reading exercise.
  • How often: Run it once today, then repeat 3-7 times before judging results.
  • Common mistake: Changing multiple things at once. Use one tool at a time so you can see what works.
  • If it feels awkward: Use the script exactly once, keep the first run under 10 minutes, and rate helpfulness 1-10.
