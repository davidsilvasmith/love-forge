10-Minute Weekly Meeting

For busy parents • practical in 10-15 minutes

🕒 10 minutes 🗓️ Weekly cadence 🛡️ Low-drama format 📈 Measurable progress
Problem

No maintenance rhythm

Without a weekly check-in, tension compounds and small issues become fights.

Outcome

One clear weekly reset

You prevent drift with one short structure that actually survives busy weeks.

Constraint

Keep it tiny

If it feels heavy, you will skip it. Keep the meeting short and repeatable.

Quick start (tonight)

  1. Schedule a recurring 10-minute Sunday check-in.
  2. Start a timer before you begin.
  3. Use the script exactly (no freestyle under stress).
  4. End with one tiny action for tomorrow.

Use this exact plan (copy this)

  1. Minute 0-2: Name today’s focus in one sentence.
  2. Minute 2-6: Each person gets 2 minutes uninterrupted.
  3. Minute 6-9: Choose one action for next 24 hours.
  4. Minute 9-10: Confirm when you’ll check in again.

Copy/paste opener

“I don’t want a giant relationship talk. I want one small win tonight. Can we try the 10-Minute Weekly Meeting for 10 minutes and see if it helps?”

Normal week

Full 10-minute pass

Run the full sequence: check-in, one tension, one appreciation, one next step.

Kid-chaos week

5 + 5 split mode

Do five minutes now and five later. Incomplete is still better than skipped.

Emotionally flooded

Pause-first mode

Run Pause and Return first, then finish when calm.
