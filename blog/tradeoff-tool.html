Tradeoff Tool

For busy parents • practical in 10-15 minutes

📵 Reduce phone drift ⏱️ 10-15 minutes 💞 One nightly win 📈 Track score 1-10
Problem

Closeness loses to default habits

Schedules and scrolling consume the energy you meant to give each other.

Tradeoff

15 minutes less noise

Swap one low-value habit for one protected connection block tonight.

Result

Lower friction, faster repair

Small reps reduce escalation and make hard talks easier later.

Quick start (tonight)

  1. Choose one tradeoff: 15 fewer scrolling minutes for 15 connection minutes.
  2. Set a timer so the conversation stays light and contained.
  3. Use the script exactly once, no freestyle.
  4. End with one specific next step for tomorrow.

Copy/paste script

“I don’t want a giant relationship talk. I want one small win tonight. Can we try the Tradeoff Tool for 10 minutes and see if it helps?”

When exhausted

Kid-chaos mode

Do it after bedtime, on a walk, or in the pickup line. Keep it short.

When interrupted

Chunk mode

Split into two 5-minute rounds instead of canceling entirely.

When heated

Reset mode

Run Pause-and-Return Tool first, then finish this tool calm.

7-day scoreboard

🔥 Fewer escalations 🩹 Faster repair ❤️ +2 connection points

Rule: Keep tools that improve your score by 2+ points. Swap the rest.

If one partner says “not now”

  • “Totally fair. What time tonight can we give this 10 minutes?”
  • If not tonight: “Let’s lock 10 minutes tomorrow so this doesn’t drift.”
  • If either person is flooded: use Pause-and-Return Tool, then return.

Goal: no blame, no pressure — just a specific next time on calendar.
