Problem

Stress kills good intentions

When energy is low, couples fall back to old patterns and react on autopilot.

Goal

Decide before the trigger

Use one if/then rule so your next move is pre-picked, not improvised.

Constraint

Keep it short + doable

Timer on. One script. One next step. No giant relationship summit.

Quick start (tonight)

⚠️

Name one trigger

Pick the moment that usually derails you (late-night logistics, money tension, tone spiral).

✍️

Write one if/then rule

Example: “If voices rise, then we pause for 10 minutes and restart with one sentence each.”

🗣️

Read the script exactly

No freestyle. Keep words simple so both people feel safe enough to try.

✅

Lock tomorrow’s action

Choose one tiny follow-through move so this becomes a pattern, not a one-off.

Copy/paste script

“I don’t want a giant relationship talk. I want one small win tonight. Can we try the Pre-Commitment Tool for 10 minutes and see if it helps?”

Starter kit (no guesswork)

Pick one of these and run it exactly tonight:

  1. 5-minute version: Read the copy/paste script out loud, then do one tiny action from this tool immediately.
  2. 10-minute version: Set a timer, follow the “Do this tonight” steps top to bottom, and stop when timer ends.
  3. Couple-text version: Send this now: “Can we do Pre-Commitment Tool tonight for 10 minutes? I already picked the first step so this is easy.”

If you feel stuck, use the 5-minute version first. Progress beats perfection.

How to use this tool effectively

  • When to use it: Use Pre-Commitment Tool when this exact problem is happening now, not as a general reading exercise.

  • If it feels awkward: Use the script exactly once, keep the first run under 10 minutes, and rate helpfulness 1-10.
