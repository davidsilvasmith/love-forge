Marriage-as-a-Choice Tool

For busy parents • practical in 10-15 minutes

💞 Rebuild couple energy ⏱️ 10-minute reset 📅 Calendar one action 🧪 7-day test

Quick start block (tonight)

1️⃣

Set the timer

Lock 10 minutes after kids are down.

2️⃣

Each answer one prompt

"What is one small way we can choose us this week?"

3️⃣

Pick and schedule

Choose one action under 20 minutes and put it on calendar now.

The real-life problem

You became a logistics team

Kids, meals, work, sleep. Love is still there, but "us" time disappears.

Simple rule

One small win beats perfect

Don’t chase a huge talk. One intentional moment this week is enough.

Sample agenda (copy this)

  1. Minute 0-2: Say what you want to improve this week.
  2. Minute 2-6: Each person speaks for 2 minutes uninterrupted.
  3. Minute 6-9: Pick one concrete action for tomorrow.
  4. Minute 9-10: Confirm check-in time and appreciation.

Copy/paste opener

“I miss feeling like we’re more than a to-do team. Can we take 10 minutes tonight and pick one small way to choose us this week?”

Pick tonight’s version

Normal energy

Full 10-15 minute run

Do the full prompt, choose one action, schedule it before ending.

Kid-chaos / exhausted

Split into 2 x 5 minutes

Do prompt now, pick + schedule later tonight while doing chores.

Too heated to talk

Use Pause-and-Return first

De-escalate first, then run this tool when calm.
