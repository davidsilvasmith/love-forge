Disconnection Detector

For busy parents • practical in 10-15 minutes

💬 Conversations feel flat ⚡ Small things turn into fights 📵 Teamwork feels off 🧊 Affection dropped
Problem

You feel off, but can’t name it

That uncertainty creates friction and distance fast.

Move

Run one 10-minute scan tonight

Measure closeness, affection, teamwork, and fun on a 1–10 scale.

Win

Get one tiny improvement by tomorrow

Pick one action, test it, then score if it worked.

Quick start (tonight)

  1. Set a 10-minute timer.
  2. Each partner scores: closeness, affection, teamwork, fun (1–10).
  3. Pick one weak area and one tiny action for tomorrow.

Script: “I don’t want a giant relationship talk. I want one small win tonight. Can we do a 10-minute Disconnection Detector and see if it helps?”

Choose your mode

Mode A

Low energy / exhausted

Do 2 scores only: closeness + teamwork. Pick one 5-minute action.

Mode B

Normal evening

Run all 4 scores. Pick one action and schedule it for tomorrow.

Mode C

Conflict is active

Pause this tool. Use Pause-and-Return first, then come back when calm.

What success looks like in 7 days

📉 Fewer escalations 🛠️ Faster repair 📈 +2 points in connection score

Rule: Keep tools that improve score by 2+ points. Swap anything that stalls.
