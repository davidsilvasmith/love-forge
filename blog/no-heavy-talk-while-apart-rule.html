No-Heavy-Talk-While-Apart Rule

For busy parents • protect connection while apart

📵 No conflict spirals by text 🧠 Lower stress while apart 🤝 Better reunion conversations ⏱️ Set up in 10 minutes

Quick start (do this tonight)

Step 1

Pick the boundary

No heavy topics over text/video unless urgent safety or logistics.

Step 2

Use the script once

Say the exact line below. No debating the rule in the moment.

Step 3

Schedule reunion talk

Lock a concrete time to discuss in person and protect trust.

Use this exact plan (copy this)

  1. Minute 0-2: Agree this is a defer-and-protect conversation.
  2. Minute 2-5: Use the script and set a reunion talk time.
  3. Minute 5-8: Confirm urgent vs non-urgent topics.
  4. Minute 8-10: End with one reassurance line.

Copy/paste script

“I care about us too much to do a heavy conversation over text while we’re apart. Let’s park this and talk in person at [time] so we can do it right.”

Choose your mode

Calm but tense

Defer + schedule

Use the script, set time, then switch to logistics only.

Kid-chaos / low energy

2 x 5-minute setup

Agree to the boundary now, pick reunion talk slot later tonight.

Flooded or escalating

Pause-and-Return first

If either person is flooding, de-escalate first, then schedule.
