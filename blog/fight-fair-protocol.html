12. Fight Fair Protocol

Problem-first framework

Problem

Arguments escalate fast and become personal. You both leave hurt, not heard.

Why this happens

  • Defensiveness feels safer than vulnerability.
  • Old unresolved issues get dragged into new conflicts.
  • No shared rules means each argument becomes a street fight.

Use this protocol every conflict

  1. Name one issue only: "I want to solve X, not everything."
  2. Use clean complaint format: "When ___ happened, I felt ___. I need ___."
  3. Reflect before rebuttal: other person must summarize first.
  4. Own your percentage: each partner states 10-20% they own.
  5. Make one concrete agreement for next 48 hours.

Banned moves

  • "You always/you never"
  • Name-calling, contempt, eye-rolling
  • Threats (divorce, leaving) to win a point
  • Bringing up old closed cases

Pause-and-return safety valve

If either person is emotionally flooded (heart racing, shouting, shutdown), call: "Pause. I will return at [time]." Return within 20-60 minutes. Not returning is not allowed.

Copy/paste script

"I’m not trying to beat you. I’m trying to solve this with you. Can we do one issue, one ask, one agreement?"

"Here’s my 20%: _____."

Success metric

For the next 2 weeks, track: number of escalations, average conflict length, and time-to-repair. Keep if escalations drop by 30%+.
