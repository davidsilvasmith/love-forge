Problem

Talks get mean, fast

When you are tired and rushed, small things turn into big fights.

Promise

One simple way to fight cleaner

You can handle one hard topic tonight without tearing each other down.

Do this tonight (10-15 minutes)

  1. Pick one issue only.
  2. Set a 12-minute timer.
  3. Use the script below, then take turns.
  4. End with one small step for tomorrow.

Copy/paste script: “I love you. I don’t want another huge fight. Can we take 10 minutes, stick to one topic, and leave with one small next step?”

Use this exact plan (copy this)

  1. Minute 0-2: Name today’s focus in one sentence.
  2. Minute 2-6: Each person gets 2 minutes uninterrupted.
  3. Minute 6-9: Choose one action for next 24 hours.
  4. Minute 9-10: Confirm when you’ll check in again.

How to use this tool effectively

  • When to use it: Use Fight Fair Tool when this exact problem is happening now, not as a general reading exercise.
  • How often: Run it once today, then repeat 3-7 times before judging results.
  • Common mistake: Changing multiple things at once. Use one tool at a time so you can see what works.
  • If it feels awkward: Use the script exactly once, keep the first run under 10 minutes, and rate helpfulness 1-10.
