Ownership Ratio Tool

For busy parents • practical in 10-15 minutes

The real-life problem

You both feel blamed, so both of you get defensive.

When life is kids + work + zero sleep, this happens a lot. You are not broken.

Do this tonight (10-15 minutes)

  1. Set a 10-minute timer.
  2. Person A says: “Here’s my part.” Name one thing you did that made it worse.
  3. Person B does the same. No interrupting. No fixing yet.
  4. Each person says one small thing they will do tomorrow.

Copy/paste script

“I love you. I don’t want a big fight tonight. Can we do a 10-minute reset and each own our part?”

Kid-chaos version (when you’re exhausted)

  • Do it in tiny chunks: 5 minutes now, 5 minutes later.
  • Use voice notes if you can’t talk live.
  • If either person is too heated, pause and come back after 20 minutes.

What “better” looks like in 7 days

  • Fights end sooner
  • Less blame, more teamwork
  • You feel calmer in the same hard moments

Keep it simple: If this helps even a little, keep using it this week.
