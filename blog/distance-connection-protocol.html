34. Distance Connection Protocol

Problem-first framework

Problem

When one partner is traveling, communication turns into logistics and emotional drift starts fast.

Why this happens

  • Different schedules create missed expectations.
  • Text strips tone and increases misreads.
  • No plan means each person guesses what “enough contact” looks like.

Set this before the trip (5 minutes)

  1. Agree one daily call window (10-12 minutes).
  2. Set expectation for response lag (example: "I may reply within 2-4 hours").
  3. Pick one reunion ritual for return day.

Daily structure while apart

  1. Morning: one supportive text (not logistics).
  2. During day: one photo/voice note from real life.
  3. Evening: 10-12 minute call: feelings first, logistics last.

Call agenda (12 minutes)

  1. "How are you, really?" (4 min)
  2. One highlight + one stress point (4 min)
  3. One appreciation + one “looking forward to” for reunion (4 min)

Avoid this while apart

  • Heavy unresolved conflict by text
  • Interrogation-style check-ins
  • Keeping score on response speed

Copy/paste script

"I want us to feel close while we’re apart. Can we do a 12-minute no-phone call tonight—feelings first, logistics last?"

Success metric

Rate connection daily 1-10. If average stays 7+ during travel, protocol is working. If not, shorten texts and increase voice/video.
