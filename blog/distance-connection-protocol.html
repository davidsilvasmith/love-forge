Distance Connection Protocol

For busy parents • practical in 10-15 minutes

Do this first tonight

If you are apart and starting to feel like coworkers, not partners, do not wing this call. Pick one focus, copy one text, and run one short rep.

💞 Less drift
🧯 Fewer blowups
📱 One exact text
📈 Better connection score
  1. Pick one fixed 10-12 minute call window.
  2. Pick one focus only: connection, logistics, or repair.
  3. Send the exact text below.
  4. Use a timer and stop after one rep.

Copy/paste starter text: “I do not want a giant relationship talk tonight. Can we do one 10-minute distance check-in at [time] and keep it to one focus only?”

Low-energy fallback: “I am wiped, but I do not want us to drift. Can we do 5 minutes tonight and the full check-in tomorrow at [time]?”

Pick ONE focus in 20 seconds

If more than one thing is wrong, stop at the first match:

  1. Repair first: use repair if the last call ended tense, cold, or unfinished.
  2. Logistics second: use logistics if travel, pickups, money, or schedule confusion is creating friction.
  3. Connection third: use connection if nothing is on fire but you feel far apart.

Say this out loud: “Tonight we are doing [one focus] only. Everything else waits until tomorrow or the weekly meeting.”

Problem

Travel creates emotional distance fast

Separate schedules, kid logistics, and short texts can make you feel like roommates with admin duties.

Promise

One repeatable 10-minute reset

Short, scripted daily check-ins keep the connection warm without turning every call into homework.

Use this exact plan (copy this)

  1. Minute 0-2: Name tonight’s one focus in one sentence.
  2. Minute 2-6: Each person gets 2 minutes uninterrupted.
  3. Minute 6-8: Reflect back one thing you heard right.
  4. Minute 8-10: Choose one action for the next 24 hours and confirm the next check-in time.

How to use this tool effectively

  • When to use it: Use Distance Connection Protocol when travel, opposite schedules, or nights apart are creating drift right now.
  • How often: Run it once tonight, then repeat 3-7 times before judging it.
  • Common mistake: Mixing logistics, resentment, and repair into the same call. Pick one focus only.
  • If it feels awkward: Use the script exactly once, keep the first run under 10 minutes, and rate helpfulness 1-10.

Tomorrow rep + fallback + escalation

  • Tomorrow: run this same tool once more before trying a new one.
  • If it didn’t help: switch to Pause-and-Return Tool first, then retry.
  • If it got worse or felt unsafe: stop DIY and involve a licensed professional/support resource.
