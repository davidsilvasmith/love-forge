Distance Connection Protocol

For busy parents • practical in 10-15 minutes

💞 Less drift
🧯 Fewer blowups
🔁 Faster repair
📈 Better connection score
Problem

Travel creates emotional distance

Separate schedules, kid logistics, and short texts can make you feel like roommates.

Promise

One repeatable 12-minute reset

Short, scripted daily check-ins keep the connection warm while life is chaotic.

Quick start block (tonight)

  1. Pick one fixed 12-minute call window.
  2. Use a timer so it stays light.
  3. Run one script exactly (no freestyle).
  4. End with one concrete step for tomorrow.

Copy/paste script: “I don’t want a giant relationship talk. I want one small win tonight. Can we run the Distance Connection Protocol for 10 minutes and see if it helps?”

Use this exact plan (copy this)

  1. Minute 0-2: Name today’s focus in one sentence.
  2. Minute 2-6: Each person gets 2 minutes uninterrupted.
  3. Minute 6-9: Choose one action for next 24 hours.
  4. Minute 9-10: Confirm when you’ll check in again.

Kid-chaos version

  • Do it after bedtime, during pickup line, or while walking.
  • If interrupted, split into two 5-minute chunks.
  • If tension spikes, pause and return before finishing the script.

Scoreboard (7 days)

How to use this tool effectively

  • When to use it: Use Distance Connection Protocol when this exact problem is happening now, not as a general reading exercise.
  • How often: Run it once today, then repeat 3-7 times before judging results.
  • Common mistake: Changing multiple things at once. Use one tool at a time so you can see what works.
  • If it feels awkward: Use the script exactly once, keep the first run under 10 minutes, and rate helpfulness 1-10.
