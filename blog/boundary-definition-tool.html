18. Boundary Definition Tool

Problem-first framework

Problem

You haven’t clearly defined emotional or digital boundaries.

What this looks like

  • Repeated friction around the same pattern
  • Lower emotional safety and warmth
  • No shared system to correct course

Tool: How to run it

  1. Name the problem in one sentence, no blame.
  2. Agree a 10–20 minute time box.
  3. Use one script + one measurable action this week.
  4. Review results in your weekly check-in.

Script starter

"I want us to win. The pattern I see is: _____. Can we try Boundary Definition Tool for 7 days and review on Sunday?"

Success metric

Rate this area 1–10 today and again in 7 days. Keep if score improves by 2+ points.
