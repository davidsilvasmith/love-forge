18. Boundary Definition Tool

Problem-first framework

Problem

Trust erodes when boundaries are implied, inconsistent, or only discussed after damage.

Define boundaries in 15 minutes

  1. Each person writes 3 lines they consider "over the line" (digital + in-person).
  2. Compare lists and combine into one shared boundary agreement.
  3. Define what happens if a boundary is crossed (disclose within 24h, repair plan).

Starter categories

  • Private emotional intimacy with someone else
  • Deleted/hidden messages and secrecy patterns
  • Flirtation in DMs/comments/work travel contexts

Copy/paste script

"I want us to protect trust proactively, not reactively. Can we define 3 clear boundaries each and agree on them tonight?"

Success metric

Rate trust 1-10 weekly. If trust rises and boundary arguments drop for 4 weeks, the agreement is working.
