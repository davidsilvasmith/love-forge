Micro-Betrayal Audit

For busy parents • practical in 10-15 minutes

The real-life problem

Trust gets chipped away by little letdowns.

Stuff like “I’ll be home by 6,” “I’ll text when I leave,” or “I’ll handle bedtime” sounds small, but it adds up fast.

Do this tonight (10-15 minutes)

  1. Each person names 1 small letdown from this week (keep it short and specific).
  2. The other person says: “You’re right. I missed that. I’m sorry.” No defending.
  3. Pick 1 repair you can do in the next 24 hours (send update text, take a chore, be on time).
  4. Set a simple check-in time for tomorrow: “Did we do what we said?”

Copy/paste script

“Can we do a quick 10-minute trust reset tonight? One small miss, one real apology, one fix by tomorrow. I want us to feel like a team again.”

Kid-chaos version (when you’re exhausted)

  • Do it while cleaning up the kitchen or getting lunches ready.
  • If a kid interrupts, finish just one round: one miss + one apology + one fix.
  • If either of you gets heated, pause for 20 minutes and come back.

Quick action checklist (easiest version)

  1. Name one small miss from this week.
  2. Give one clear apology with no defense.
  3. Pick one repair to complete in 24 hours.
  4. Confirm tomorrow’s check-in time.

Tool focus: Micro-Betrayal Audit

What “better” looks like in 7 days

  • Fewer “you said you would” fights
  • More follow-through on small promises
  • You both feel safer asking for what you need

Rule: Keep this tool if follow-through improves this week. If not, switch tools.

How to use this tool effectively

  • When to use it: Use Micro-Betrayal Audit when this exact problem is happening now, not as a general reading exercise.
  • How often: Run it once today, then repeat 3-7 times before judging results.
  • Common mistake: Changing multiple things at once. Use one tool at a time so you can see what works.
  • If it feels awkward: Use the script exactly once, keep the first run under 10 minutes, and rate helpfulness 1-10.
