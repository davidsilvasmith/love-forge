Ritual of Reunion Tool

For busy parents • zero-guesswork reunion plan

The real-life problem

After work/school chaos, evenings start cold and transactional.

A 3-minute reunion ritual resets tone before logistics take over.

Use this exact 3-minute ritual (copy this)

  1. 0:00-0:20 — Touch: 20-second hug or hand squeeze, no phones.
  2. 0:20-1:20 — One line each: “My energy right now is ___/10.”
  3. 1:20-2:20 — Appreciation: “Thanks for ___ today.”
  4. 2:20-3:00 — Next step: “Tonight, one thing that would help me is ___.”

Copy/paste script

“When we reconnect tonight, can we do a 3-minute reset first? It’ll help me feel like we’re on the same team before we jump into logistics.”

Starter kit (no guesswork)

  1. Tonight: Use the exact 3-minute ritual above once.
  2. Tomorrow: Repeat same ritual at same reunion moment.
  3. After 3 days: Keep what felt good; swap one awkward step.

Don’t invent from scratch. Run the script as written, then iterate.

Kid-chaos fallback

  • If kids interrupt: do 60-second mini-version (touch + appreciation + one ask).
  • If one partner is flooded: run Pause-and-Return first.
  • If timing keeps failing: move ritual to a predictable anchor (bedtime kitchen cleanup).

How to use this tool effectively

  • When to use it: Daily reunion moments (work return, pickup, bedtime handoff).
  • How often: 5 nights this week minimum.
  • Common mistake: Turning this into a problem-solving meeting.
  • If awkward: Keep it to 90 seconds for 3 days, then expand.

Tomorrow rep + fallback + escalation

  • Tomorrow: run this same tool once more before trying a new one.
  • If it didn’t help: switch to Pause-and-Return Tool first, then retry.
  • If it got worse or felt unsafe: stop DIY and involve a licensed professional/support resource.
