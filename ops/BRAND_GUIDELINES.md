# Love Forge Brand Guidelines

## Brand promise
Practical relationship help that feels hopeful, honest, and usable tonight.

## Voice
- Warm, grounded, non-judgmental
- Plain language (no therapy jargon unless explained)
- Specific > abstract
- Helpful before salesy

## Visual direction
- Emotion-first but calm
- Modern, premium, clean
- Use color and icons to reduce text fatigue
- Mobile-first readability

## Content standards
Each page should include:
1. Problem in human language
2. Why it happens
3. 10-15 minute action
4. Copy/paste script
5. Simple success metric

## CTA standards
- Primary CTA = next best helpful step (quiz or start tool)
- Secondary CTA = browse tool library
- No aggressive urgency language


## Primary audience anchor
Write for a 35-year-old busy parent (soccer-mom readability): short, practical, emotionally safe, no jargon, no lectures.
