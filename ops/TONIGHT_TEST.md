# Tonight Test (Pre-Ship Checklist)

Use this before shipping any page.

## Pass/Fail Questions

1. In 10 seconds, can a user tell what to do first?
2. In 60 seconds, can a user copy one script and start?
3. If they’re exhausted, is there a low-energy version?
4. If emotions spike, is there a fallback path?
5. Is there a concrete 7-day success signal?
6. If multiple issues are likely, does the page explicitly say how to pick ONE issue tonight?

If any answer is NO -> do not ship.

## Required blocks by page type

### Conversational tools
Must include:
- Exact plan (minute-by-minute)
- Copy/paste script
- Repair fallback

### Habit/setup tools
Must include:
- Starter kit checklist
- Copy/paste script
- Tiny first rep for tonight

### Complex tools
Must include at least one:
- Builder
- Template
- Fill-in form users can copy

## Anti-patterns (hard fail)
- “Create your own …” with no template
- “Talk about …” with no script
- “Set boundaries” with no boundary examples
- “Check in weekly” with no agenda
